## 💡 ИДЕЯ

Мобильное приложение для быстрого расчёта стоимости на месте у клиента.

Замерщик приезжает → открывает приложение → вводит данные → получает цену → показывает клиенту прямо на экране.

## 🧮 КАК СЧИТАЕТ ЦЕНУ

```
Площадь = Ширина × Высота (в sq ft)
Цена материала = площадь × material_cost (из прайса)
Цена работы = площадь × labor_cost (из прайса)
Сложность = фиксированная надбавка
Срочность = % надбавка на итог
Доп. работы = фиксированные суммы из чеклиста
Итог = материал + работа + сложность + срочность + доп.работы
Tax = итог × 10.25% (LA County tax)
```

## 🔮 ДОПОЛНИТЕЛЬНЫЕ ИДЕИ (записать, не забыть)

1. **Нарисовать стекло** — простой редактор где можно нарисовать форму нестандартного стекла (арки, треугольники)

2. **Фото → AI размер** — сфоткал окно, AI определяет примерный размер

3. **Голосовой ввод** — надиктовал размеры, AI распознал и заполнил поля

4. **История объекта** — введи адрес, видишь все предыдущие работы на этом адресе

5. **Подпись клиента** — клиент подписывает пальцем прямо на экране = согласие

6. **Stripe оплата** — клиент платит депозит прямо на месте
