# ROMI CRM

> Glass & Window Repair CRM — Built for the US market.
> American alternative to Ramex, designed specifically for glass repair companies.

## Vision
A specialized CRM for glass and window repair companies in the USA.
Inspired by Ramex (Russia), built for the American market with Twilio, CallRail, and AI Agent integration.

## Tech Stack (planned)
- Backend: Python + FastAPI
- Database: PostgreSQL
- Frontend: Next.js + Tailwind CSS
- Mobile: React Native (for field technicians)
- Telephony: Twilio
- Call Tracking: CallRail
- AI: OpenAI (GPT + Whisper)

## Core Modules
1. Leads & Contacts
2. Pipeline (Call → Measure → Quote → Contract → Install → Paid)
3. Measure Tool (field app for technicians)
4. Price Calculator (glass types + dimensions)
5. Quote Builder (PDF generator)
6. Financials (cost + labor + profit)
7. Team Management (GPS tracking)
8. Claims & Warranty
9. Integrations (Twilio, CallRail, Google Ads)
10. AI Agent (24/7 voice + SMS)
